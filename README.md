AI Debate Coach
**Overview**
AI Debate Coach is an intelligent, real-time debate training platform that helps users improve their speaking, argumentation, and confidence.

**It provides:**
Real-time counter-arguments
Personalized coaching feedback
Performance tracking
**Problem Statement**

Students and professionals often struggle with:
Lack of confidence in public speaking
Weak argument structuring
No real-time feedback while practicing
This project solves these issues by combining AI + Speech + Computer Vision into one interactive system.

**Features**
1 Speech-to-Text Input using Web Speech API
2 AI Debate Opponent using Gemini API
3 Smart Coaching System for feedback and improvement
4 Eye Contact Detection using MediaPipe
5 Performance Analysis (fluency, confidence, behavior)
6 Real-time Interaction
7 Web-based UI

**Tech Stack**
Frontend--React / HTML / CSS / JavaScript, Web Speech API, MediaPipe (Computer Vision)
Backend--Flask (Python), REST APIs, Grok AI Integration
AI / ML--Grok Api
